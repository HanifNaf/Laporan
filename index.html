How to Create Github Page Using Git


About

Github is used by many people to host their work. Some people host their projects, some just want to use github to backup their files. But it turns out, Github can be used to do something else. You can use Github to host your own website. It is free and fairly easy. In this article I will show you how to host a Github page using Git.



1. Create a Github account and create new repository



The first step is to set-up your Github account. This can be done by clicking the sign up button and following the direction it gives. After you successfully create your account, create a repository to set-up your web page.


Name your repository with ['your account name'.github.io] this will ensure that your web is running on the main branch (not a subdirectory).



2. Go to your local folder and initiate git



After the repository is set, go to the folder which contains your file. Right click on it and click the git bash here button. (install git here)


After the git window showed up, type git init and press enter.



3. Add your files to the staging area



Type git add . then press enter to add your files to the staging area. Dont forget to include the "dot" so that all your files are added.



4. Commit your files



Type git commit -m "first commit" then press enter to commit your files.



5. Add remote origin



Go to your repository and copy its link.


Type git add remote origin followed by the link to your repository.



6. Push your files




The last step is to push your file to the repository. Type git push origin master and press enter.


Congratulations. Now your website is ready and you can always edit it by following the same steps
